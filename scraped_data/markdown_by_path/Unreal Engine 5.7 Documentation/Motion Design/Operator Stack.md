# Operator Stack

Use the Operator Stack window to access Modifiers and Animators for your Motion Design project.

![Operator Stack](https://dev.epicgames.com/community/api/documentation/image/04e13881-78bf-46ab-bb13-b635deacd393?resizing_type=fill&width=1920&height=335)

 On this page

The **Operator Stack** provides access to two different suites of tools for working with Motion Design projects, Modifiers and Animators.

* The **Modifiers** include a variety of tools for working with both 2D and 3D projects, including modifying layouts, geometry, and rendering, as well as a modifier to assist with using transition logic.
* The **Animators** include tools to animate your designs without entirely relying on keyframing, using a variety of customizable animation presets.

## Where to Find the Tools

You can access the Operator Stack in two ways after you activate Motion Design Mode.

[![Motion Design Mode](https://dev.epicgames.com/community/api/documentation/image/daa2112a-2fd9-43f7-82a5-c0f30119d455?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/daa2112a-2fd9-43f7-82a5-c0f30119d455?resizing_type=fit)

1. Click **Operator Stack** on the left of the main **Motion Design toolbar**:

   [![Operator Stack button](https://dev.epicgames.com/community/api/documentation/image/8eb452c6-fbd0-4cac-9670-fca847f0a479?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/8eb452c6-fbd0-4cac-9670-fca847f0a479?resizing_type=fit)
2. Open it from the **Window** menu:

   [![Operator Stack in the Window menu.](https://dev.epicgames.com/community/api/documentation/image/1bb15145-e5db-423a-adb9-87de8aef63d1?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/1bb15145-e5db-423a-adb9-87de8aef63d1?resizing_type=fit)

## Operator Stack Window

After opening the Operator Stack, a window appears in your interface that resembles the following image:

[![Operator Stack window](https://dev.epicgames.com/community/api/documentation/image/d519e0bf-ecb8-4dc7-b41f-8d8e6b67041a?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/d519e0bf-ecb8-4dc7-b41f-8d8e6b67041a?resizing_type=fit)

The tab on the left in the Operator Stack window is for working with modifiers. The tab on the right is for working with animators.
