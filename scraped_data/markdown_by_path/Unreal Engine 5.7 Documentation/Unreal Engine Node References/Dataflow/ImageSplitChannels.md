### Description

ImageSplitChannels (v1)

Split a image in individual channels
Outputs are single channel images

Input(s) :
Image - Input image to split per channel

Output(s):
Red - Red channel
Green - Green channel
Blue - Blue channel
Alpha - Alpha channel
