### Inputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Seed | The seed for the random number generator | [int32](/documentation/en-us/unreal-engine/API/Runtime/Core) | 217 |
| Minimum | The minimum value for the random number range | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) | 0.000000 |
| Maximum | The maximum value for the random number range | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) | 1.000000 |
| Duration | The duration at which the number won't change. Use 0 for a different number every time. A negative number (for ex: -1) results in an infinite duration. | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) | 0.000000 |
