# Get Control Float

GetControlFloat is used to retrieve a single Float from a hierarchy.

On this page

### Description

GetControlFloat is used to retrieve a single Float from a hierarchy.

### Information

|  |  |
| --- | --- |
| Plugin | [ControlRig](/documentation/en-us/unreal-engine/API/PluginIndex/ControlRig) |
| Category | Controls |
| Tags | GetControlFloat |
| Type | [FRigUnit\_GetControlFloat](/documentation/en-us/unreal-engine/API/Plugins/ControlRig/FRigUnit_GetControlFloat) |

### Inputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Control | The name of the Control to retrieve the Float for. | [FName](/documentation/en-us/unreal-engine/API/Runtime/Core/FName) | None |

### Outputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| FloatValue | The current value of the control. | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) |  |
| Minimum | The minimum value of the control. | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) |  |
| Maximum | The maximum value of the control. | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) |  |
