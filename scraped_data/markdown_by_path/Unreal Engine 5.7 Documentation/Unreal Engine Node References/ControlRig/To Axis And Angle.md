# To Axis And Angle

Retrieves the axis and angle of a quaternion in radians

On this page

### Description

Retrieves the axis and angle of a quaternion in radians

### Information

|  |  |
| --- | --- |
| Plugin | [RigVM](/documentation/en-us/unreal-engine/API/PluginIndex/RigVM) |
| Category | Math|Quaternion |
| Tags | Make,Construct,GetAxis,GetAngle |
| Type | [FRigVMFunction\_MathQuaternionToAxisAndAngle](/documentation/en-us/unreal-engine/API/Plugins/RigVM/FRigVMFunction_MathQuaternionToA-) |

### Inputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Value | The input quaternion | [Quat](/documentation/en-us/unreal-engine/API/Runtime/Core) | (X=0.000000,Y=0.000000,Z=0.000000,W=1.000000) |

### Outputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Axis | The axis of the quaternion | [Vector](/documentation/en-us/unreal-engine/API/Runtime/Core) |  |
| Angle | The angle in radians of the quaternion | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) |  |
