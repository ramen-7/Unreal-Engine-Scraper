# Curve Exists

CurveExists is used to check whether a curve exists or not.

On this page

### Description

CurveExists is used to check whether a curve exists or not.

### Information

|  |  |
| --- | --- |
| Plugin | [ControlRig](/documentation/en-us/unreal-engine/API/PluginIndex/ControlRig) |
| Category | Curve |
| Tags | CurveExists,bool |
| Type | [FRigUnit\_CurveExists](/documentation/en-us/unreal-engine/API/Plugins/ControlRig/FRigUnit_CurveExists) |

### Inputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Curve | The name of the Curve to retrieve the transform for. | [FName](/documentation/en-us/unreal-engine/API/Runtime/Core/FName) | None |

### Outputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Exists | Boolean indicating whether the named curve exists or not. Does not indicate whether the curve's value is valid or not. | [bool](/documentation/en-us/unreal-engine/API/Runtime/Core) |  |
