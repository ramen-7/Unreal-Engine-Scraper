# Parameter At Length Percentage

Returns the U parameter of a spline given a length percentage (0.0 - 1.0)

On this page

### Description

Returns the U parameter of a spline given a length percentage (0.0 - 1.0)

### Information

|  |  |
| --- | --- |
| Plugin | [ControlRigSpline](/documentation/en-us/unreal-engine/API/PluginIndex/ControlRigSpline) |
| Category | Splines |
| Type | [FRigUnit\_ParameterAtPercentage](/documentation/en-us/unreal-engine/API/Plugins/ControlRigSpline/FRigUnit_ParameterAtPercentage) |

### Inputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| Spline | The spline to evaluate | [Control Rig Spline](/documentation/en-us/unreal-engine/API/Plugins/ControlRigSpline/FControlRigSpline) | () |
| Percentage | The percentage (0.0 - 1.0) to evaluate | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) | 0.000000 |

### Outputs

| Name | Description | Permitted Types | Default Value |
| --- | --- | --- | --- |
| U | The U value on the spline for the given percentage | [float](/documentation/en-us/unreal-engine/API/Runtime/uLangCore) |  |
