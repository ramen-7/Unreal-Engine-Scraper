### Virtual Textures

| **Section** | **Description** |
| --- | --- |
| **Auto Virtual Texturing Size** | Automatically enables the **Virtually Texture Streaming** texture setting for textures larger than or equal to this size during the texture import.  This setting does not affect existing textures in the project. |
