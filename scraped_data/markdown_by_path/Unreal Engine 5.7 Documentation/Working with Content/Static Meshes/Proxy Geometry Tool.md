# Proxy Geometry Tool

The Proxy Geometry tool set was developed as a way to increase your Unreal Engine 4 (UE4) projects performance while keeping the visual quality of your project uneffected.

![Proxy Geometry Tool](https://dev.epicgames.com/community/api/documentation/image/7c9ce4e7-56d6-41c1-9bda-c696fd29467a?resizing_type=fill&width=1920&height=335)

 On this page

The Proxy Geometry tool set was developed as a way to increase your Unreal Engine 4 (UE4) project's performance while keeping the visual quality of your project uneffected.

The Proxy Geometry Tool is currently only supported on PCs that use Microsoft Windows. Mac and Linux support will be added at a later date.
