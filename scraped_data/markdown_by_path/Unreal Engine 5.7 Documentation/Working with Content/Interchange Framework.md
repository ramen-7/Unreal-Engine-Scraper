# Interchange Framework

Information on importing and exporting content using the Interchange Framework

![Interchange Framework](https://dev.epicgames.com/community/api/documentation/image/1048ab0c-20a6-43aa-8791-47a909df552f?resizing_type=fill&width=1920&height=335)

The **Interchange Framework** is Unreal Engine's import and export framework. It is file format agnostic, asynchronous, customizable, and can be used at runtime.

[![Interchange Interface](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/d1da737f-fd93-45f7-98c8-1c62f72b4b6c/interchange-interface.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/d1da737f-fd93-45f7-98c8-1c62f72b4b6c/interchange-interface.png)

Interchange uses a code base that is extensible and provides a customizable pipeline stack. This gives you the freedom to edit the import pipeline using Blueprint or Python to fit your project's needs.
