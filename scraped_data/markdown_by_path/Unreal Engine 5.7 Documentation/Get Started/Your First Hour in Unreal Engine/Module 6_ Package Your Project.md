Module 6 Overview

In this final module, we will show you how to install Microsoft Visual Studio and all the elements you need to be able to build and package your project. Once Visual Studio is installed, you will learn about the Build process and how to package your game to share with your family and friends.

### Technical Requirements

You will need Microsoft Visual Studio 2022 or higher to complete this module. To download Visual Studio, click [here](https://visualstudio.microsoft.com/downloads/).
