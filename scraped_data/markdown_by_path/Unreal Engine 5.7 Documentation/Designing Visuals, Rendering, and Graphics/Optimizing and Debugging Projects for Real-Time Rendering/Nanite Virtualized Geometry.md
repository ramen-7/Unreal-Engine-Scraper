# Nanite Virtualized Geometry

A list of topics related to using and understanding Nanite Virtualized Geometry's use in Unreal Engine.

On this page

Nanite is Unreal Engine's virtualized geometry system which uses an internal mesh format and rendering technology to render pixel scale detail and high object counts. It intelligently does work on only the detail that can be perceived and no more. Nanite's data format is also highly compressed, and supports fine-grained
