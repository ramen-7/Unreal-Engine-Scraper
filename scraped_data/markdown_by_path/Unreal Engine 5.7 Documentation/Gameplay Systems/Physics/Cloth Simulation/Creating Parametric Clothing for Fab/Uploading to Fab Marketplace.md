# Uploading to Fab Marketplace

Uploading your MetaHuman to Fab Marketplace.

![Uploading to Fab Marketplace](https://dev.epicgames.com/community/api/documentation/image/701dff7f-f3d8-4e58-8628-c58ff6bc5623?resizing_type=fill&width=1920&height=335)

 On this page

If you are converting an asset from an existing **Fab** listing, keep your existing listing and add an additional **MetaHuman** format to your existing listing. There is no need to create a new listing, and this way, buyers who have purchased your items previously will automatically have access to the new MetaHuman format version of your asset. If this item is not already on Fab, you can create a new listing.

## Create a New Listing on Fab

1. Add a MetaHuman Format to the listing.

   [![MetaHuman Tag](https://dev.epicgames.com/community/api/documentation/image/8442df07-3216-4f0c-9896-abce1b025829?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/8442df07-3216-4f0c-9896-abce1b025829?resizing_type=fit)

   [![Choose MetaHuman Format](https://dev.epicgames.com/community/api/documentation/image/584521cc-49dd-40a0-97a5-9c918bb1dc5f?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/584521cc-49dd-40a0-97a5-9c918bb1dc5f?resizing_type=fit)

   [![Upload File To Fab](https://dev.epicgames.com/community/api/documentation/image/5569fe65-d1d6-42a9-a1e5-e81a8c2bac21?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/5569fe65-d1d6-42a9-a1e5-e81a8c2bac21?resizing_type=fit)
2. When filling out your listing, add the `parametric` tag for your item.

   Optionally, you can add other formats (such as Maya source files) or additional files (such as DNA files or other file types not specifically offered as options) if you want to include them in the listing.

   MetaHuman Manager will only package items in your folder which your main asset has dependencies on. If you want to include other items (such as alternate textures or materials), you must manually zip them and upload them as **Format > Other > Additional Files**.

   The `NoAI` tag is required for listings of MetaHuman-compatible content and it will be automatically applied after this format is added to the listing.

   Assets containing MetaHuman-compatible content may not be distributed under CC BY 4.0 license so this license option will be disabled for those listings.
3. Once you feel good about your listing, **Publish** it. Once it goes through the standard moderation process, it will be listed on Fab and be added to the Fab MetaHuman Channel.
