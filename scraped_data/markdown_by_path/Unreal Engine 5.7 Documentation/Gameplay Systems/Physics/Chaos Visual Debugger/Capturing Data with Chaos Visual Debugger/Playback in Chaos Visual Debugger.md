# Playback in Chaos Visual Debugger

Play back recordings in Chaos Visual Debugger.

![Playback in Chaos Visual Debugger](https://dev.epicgames.com/community/api/documentation/image/ab7918f1-2288-4d80-80f1-a546ac4266b2?resizing_type=fill&width=1920&height=335)

 On this page

**[Chaos Visual Debugger](https://dev.epicgames.com/documentation/en-us/unreal-engine/chaos-visual-debugger-in-unreal-engine)** (**CVD**) saves recordings as two types of files:

* `.utrace` (Generated by saving a single-source recording.)
* `.cvdmulti` (Generated by combining multiple `.utrace` files.)

On this page, you’ll learn how to locate and load `.utrace` and `.cvdmulti` files into CVD for:

* [Single-Source Recordings](https://dev.epicgames.com/documentation/en-us/unreal-engine/playback-in-chaos-visual-debugger#single-source-recordings)
* [Multi-Source Recordings](https://dev.epicgames.com/documentation/en-us/unreal-engine/playback-in-chaos-visual-debugger#multi-source-recordings)

## Load Single-Source Recordings

[![Single-Source Recording](https://dev.epicgames.com/community/api/documentation/image/579038b1-b22e-494d-9d3c-06b46ac84ab1?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/579038b1-b22e-494d-9d3c-06b46ac84ab1?resizing_type=fit)

CVD stores `.utrace` files in different locations on disk depending on how they were recorded and what type of application was recorded:

| Application Type | Location on Disk | Example |
| --- | --- | --- |
| * PIE session * Game client * Game server | Inside the project folder. For example:  `[ProjectFolder]/Saved/Profiling` | [PIE Session File Path](https://dev.epicgames.com/community/api/documentation/image/e98f5aa1-3c8f-4b6a-b29a-2f0758db8fbb?resizing_type=fit) |
| Packaged build | Inside the project's build folder. For example:  `[BuildFolder]/[Platform]/Saved/Profiling` | [Packaged Build File Path](https://dev.epicgames.com/community/api/documentation/image/66e7065a-715b-4c22-830f-1b611a17a29f?resizing_type=fit) |
| Live recording | Inside the Unreal Trace Folder. For example:  `UnrealEngine/Common/UnrealTrace/Store/001/` | [Live Recording File Path](https://dev.epicgames.com/community/api/documentation/image/369d4198-ec22-41b3-8dd8-79f7aa2ef531?resizing_type=fit) |

### Load a UTRACE File

To load a recording follow these steps:

1. In the main toolbar of CVD, click **Open File**.

   [![Open File](https://dev.epicgames.com/community/api/documentation/image/974b1a42-1d92-49da-8fbc-e206db7e9122?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/974b1a42-1d92-49da-8fbc-e206db7e9122?resizing_type=fit)
2. In the **Open CVD file** dialog, **Loading Mode** is set to **Single Source** by default, so you don’t need to change this option. From the **Folder** dropdown menu, select the folder your recording is stored in. Click **Open Folder**.

   [![Open CVD File](https://dev.epicgames.com/community/api/documentation/image/6cddf8b9-1017-4a9c-b827-9f304da7cfcb?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/6cddf8b9-1017-4a9c-b827-9f304da7cfcb?resizing_type=fit)
3. Select the `.utrace` file you want to load and click **Open**.

## Load Multi-Source Recordings

[![Multi-Source Recording](https://dev.epicgames.com/community/api/documentation/image/2d145bd6-fa20-438f-b416-12cec776996c?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/2d145bd6-fa20-438f-b416-12cec776996c?resizing_type=fit)

### Load Multiple UTRACE Files

Loading multiple single-source recordings is the first step to creating a CVDMULTI file. However, this can also be useful for visualizing two single-source recordings simultaneously to inspect for discrepancies.  
  
To load multiple `.utrace` files, follow these steps:

1. In the main toolbar of CVD, click Open File.

   [![Open File](https://dev.epicgames.com/community/api/documentation/image/2358c437-11c5-4482-b2ec-dde351e70de1?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/2358c437-11c5-4482-b2ec-dde351e70de1?resizing_type=fit)
2. In the **Open CVD file** dialog, in the Loading Mode dropdown, select **Multi-Source.**

   [![Open CVD File Muti Source](https://dev.epicgames.com/community/api/documentation/image/a34dcc52-6696-4d8b-a514-73aaf27531d9?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/a34dcc52-6696-4d8b-a514-73aaf27531d9?resizing_type=fit)
3. From the **Folder** dropdown menu, select the folder that contains your recording. Click **Open Folder**.

   [![Open CVD File](https://dev.epicgames.com/community/api/documentation/image/20ba525f-4bb3-4d5f-874c-f1daca760b9c?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/20ba525f-4bb3-4d5f-874c-f1daca760b9c?resizing_type=fit)
4. Select the `.utrace` files you want to load and click **Open**. CVD automatically loads both files.

### Create a CVDMULTI File

Once two `.utrace` files are loaded into CVD, the **Combine** button becomes available in the main toolbar. Click **Combine** and save the new `.cvdmulti` file at your preferred destination.

[![Combine Button](https://dev.epicgames.com/community/api/documentation/image/62c23867-2b1e-4468-b8ae-ff9e93626872?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/62c23867-2b1e-4468-b8ae-ff9e93626872?resizing_type=fit)

### Load a CVDMULTI File

To load a `.cdmulti` file, follow these steps:

1. In the main toolbar of CVD, click **Open File**.

   [![Open File](https://dev.epicgames.com/community/api/documentation/image/3c622ea3-e7cd-4be4-b356-64b4992ff39c?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/3c622ea3-e7cd-4be4-b356-64b4992ff39c?resizing_type=fit)
2. In the **Open CVD file** dialog, verify that the **Loading Mode** is set to **Single Source**. From the **Folder** dropdown menu, select the folder that contains your recording. Click **Open Folder**.

   [![Open CVD File](https://dev.epicgames.com/community/api/documentation/image/a5215d0c-79d0-4934-bd36-b0cb7daa42c9?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/a5215d0c-79d0-4934-bd36-b0cb7daa42c9?resizing_type=fit)
3. Select the `.cvdmulti` file you want to load and click **Open**.

There is a known issue where the first frame will not automatically play after the file loads. Geometry will not generate until you click the **Play** icon in the **Game Frames Timeline**.

[![Game Frames Timeline Play](https://dev.epicgames.com/community/api/documentation/image/a61b3c2b-cc53-42bb-9c91-e3a7dfe2c5ca?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/a61b3c2b-cc53-42bb-9c91-e3a7dfe2c5ca?resizing_type=fit)
