# Input

Different methods to create and setup input in Unreal Engine

![Input](https://dev.epicgames.com/community/api/documentation/image/421f1b7a-a3f2-43e8-a084-8c7d77e9e475?resizing_type=fill&width=1920&height=335)

 On this page

The **PlayerInput** Object is responsible for converting input from the player into data that Actors (like PlayerControllers or Pawns)
can understand and use. It is part of an input processing flow that translates hardware input from players into game events and movement with Player Input mappings and Input Components.

You can get started with the [Input Overview](/documentation/en-us/unreal-engine/input-overview-in-unreal-engine) or choose any of the topics below.
