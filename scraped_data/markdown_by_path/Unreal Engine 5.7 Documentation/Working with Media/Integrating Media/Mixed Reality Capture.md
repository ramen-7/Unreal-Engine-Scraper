# Mixed Reality Capture

Composite footage of users into a virtual space with Mixed Reality Capture

![Mixed Reality Capture](https://dev.epicgames.com/community/api/documentation/image/469ac4f4-9874-45c4-a45a-770155b45416?resizing_type=fill&width=1920&height=335)

Learn to use this **Beta** feature, but use caution when shipping with it.

 On this page

The Mixed Reality Capture (MRC) framework makes it easy for you to composite footage of users into a virtual space.
