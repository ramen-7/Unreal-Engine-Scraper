# Unreal Stage App

An application designed to utilize a tablet as a wireless control panel for specific Unreal Engine features in a physical space.

![Unreal Stage App](https://dev.epicgames.com/community/api/documentation/image/d1e46def-5f07-41d1-a9f2-f6e30fd107d3?resizing_type=fill&width=1920&height=335)

Learn to use this **Experimental** feature, but use caution when shipping with it.

 On this page

Explore how to use the Unreal Stage app as a wireless control panel for features in a physical space.
