# nDisplay Template

Use the nDisplay Template as the starting point for your nDisplay projects.

![nDisplay Template](https://dev.epicgames.com/community/api/documentation/image/d018f935-54c4-4a6f-a001-7fbc9d84fd90?resizing_type=fill&width=1920&height=335)

 On this page

The **nDisplay** template is the starting point for creating complex arrangements of physical displays for live performances. The template provides a variety of common nDisplay configurations to get you started.

## Creating a Project from the Template

1. Launch Unreal Engine.
2. Select the **Film, Television, and Live Events** template category.

   [![undefined](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/66f9ef8d-3864-4b74-9b45-ed4e12e527f3/01-select-template_ue5.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/66f9ef8d-3864-4b74-9b45-ed4e12e527f3/01-select-template_ue5.png)

   Click image to expand
3. Click **nDisplay**, choose whether to include starter content and whether to enable ray tracing, and select the path and name for your project. click **Next**.

   [![undefined](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/ec96e5c7-479a-4e63-90d0-1504e0ebf3ff/02-create-project_ue5.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/ec96e5c7-479a-4e63-90d0-1504e0ebf3ff/02-create-project_ue5.png)

   Click image to expand
4. Click **Create**.

   [![undefined](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/a5bc8165-2504-4396-80f3-edb7aaa0e591/02-1-create-project_ue5.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/a5bc8165-2504-4396-80f3-edb7aaa0e591/02-1-create-project_ue5.png)

   Click image to expand
