# Real-Time Compositing with Composure

In Composure, a graphics utility plugin available with Unreal Engine, we've added a new set of tools to make compositing a lot easier.

![Real-Time Compositing with Composure](https://dev.epicgames.com/community/api/documentation/image/ecc2425c-fa31-4a16-8831-15d254737d2b?resizing_type=fill&width=1920&height=335)

Learn to use this **Beta** feature, but use caution when shipping with it.

 On this page

[![](https://dev.epicgames.com/community/api/documentation/image/f65164a8-2366-42c3-a8cc-eeeed54492a8?resizing_type=fit)](https://dev.epicgames.com/community/api/documentation/image/f65164a8-2366-42c3-a8cc-eeeed54492a8?resizing_type=fit)

Composure is a graphics utility plugin for Unreal Engine that includes a set of tools for compositing. Compositing is the act of combining visual elements from different sources into a single image.

In the visual effects industry, compositing a single frame is usually an offline process and can take some time. With compositing built into the engine, users can easily visualize green-screened scenes combined with CG in real-time. This is helpful for previsualization, which gives directors a sense of what the final scene will look like, or for offline compositors to use as a guide.

Composure runs both in-editor and in-game.
