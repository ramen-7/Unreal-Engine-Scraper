# Creating Custom Modules

This page collects all the pages that will teach you how to create custom modules in Niagara.

![Creating Custom Modules](https://dev.epicgames.com/community/api/documentation/image/18712691-58a0-469a-aa0a-81edc2135ef9?resizing_type=fill&width=1920&height=335)

 On this page

Niagara gives you the full flexibility of being able to create your own custom modules. You can use the Niagara Script Editor to define your own logic flow, save your modules, and then apply them in a Niagara stack.

## Niagara Script Editor Reference

To understand how to use the Script Editor, refer to the Niagara Script Editor Reference page.

[![Niagara Script Editor Reference](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/7c14f37a-19b0-4741-907f-354d8c67f804/topic-image.png)

Niagara Script Editor Reference

This page describes the user interface (UI) of the Niagara Script Editor.](/documentation/en-us/unreal-engine/script-editor-reference-for-niagara-effects-in-unreal-engine)

## Versioning Modules in Niagara

When you are iterating on creating new versions of modules, it's a good idea to work with versioning enabled so that you don't end up breaking any scenes that are using an outdated version of your modules. This page can teach you more.

[![Versioning Modules and Emitters in Niagara](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/7154a27e-dc20-4dcb-8ac4-8005a96f1def/topic-image.png)

Versioning Modules and Emitters in Niagara

Niagara has a built-in versioning system for people that are creating their own custom modules and emitters.](/documentation/en-us/unreal-engine/versioning-modules-and-emitters-in-niagara-effects-for-unreal-engine)
