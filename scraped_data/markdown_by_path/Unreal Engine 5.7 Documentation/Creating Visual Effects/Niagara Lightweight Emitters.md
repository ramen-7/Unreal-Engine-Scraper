# Niagara Lightweight Emitters

Documentation of Niagara Lightweight Emitters

![Niagara Lightweight Emitters](https://dev.epicgames.com/community/api/documentation/image/4b4e44a9-a3af-427c-849f-088419d34dbf?resizing_type=fill&width=1920&height=335)

Niagara **lightweight emitters**, also called **stateless emitters**, are optimized to minimize (and in some cases eliminate) the use of tick during the simulation.
