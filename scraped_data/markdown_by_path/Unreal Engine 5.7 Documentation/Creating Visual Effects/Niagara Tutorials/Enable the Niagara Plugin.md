# Enable the Niagara Plugin

Describes how you can enable the Niagara Plugin.

![Enable the Niagara Plugin](https://dev.epicgames.com/community/api/documentation/image/f8d90f4a-88ef-40fd-8a53-d02e1ef4438a?resizing_type=fill&width=1920&height=335)

 On this page

In Unreal Engine 5, the Niagara plugin is installed and enabled by default. However, if you are unable to see the Niagara options then you can check to make sure the plugin is enabled.

## How to Enable the Niagara Plugin

1. First open the Plugins menu by going to **Edit** > **Plugins**.

   [![Click on Edit > Plugins](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/8ba9a5fb-2bc2-4751-a2ab-529f7c74ccff/01-edit-plugins.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/8ba9a5fb-2bc2-4751-a2ab-529f7c74ccff/01-edit-plugins.png)

   Click image for full size.
2. In the Plugins menu, locate the **FX** section. You will see Niagara listed in this section. If the checkbox is checked, Niagara is already enabled. If the checkbox is unchecked, click on it to enable Niagara.

   [![Look for Niagara in the FX section](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/ef3d23f4-bee9-4dcb-9afe-327236399a77/02-fx-plugins-niagara.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/ef3d23f4-bee9-4dcb-9afe-327236399a77/02-fx-plugins-niagara.png)

   Click image for full size.
3. If you had to enable Niagara, it will prompt you that a restart is required. Click **Restart Now** to relaunch the Unreal Editor with the Niagara plugin enabled.

   [![Restart Unreal Engine](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/e0e5b6e3-03f1-4f02-bf1f-3d108f05ce93/03-warning-restart.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/e0e5b6e3-03f1-4f02-bf1f-3d108f05ce93/03-warning-restart.png)

   Click image for full size.

## End Result

Once the UE4 Editor has restarted, when you right-click in the **Content Browser** you will have a new **FX** section that will have all of the Niagara options.

[![FX Section in the Content Browser Contextual Menu](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/32c104c7-cc89-4c7a-bb6a-9400bcaafc8d/04-niagara-in-content-browser.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/32c104c7-cc89-4c7a-bb6a-9400bcaafc8d/04-niagara-in-content-browser.png)

Click image for full size.
