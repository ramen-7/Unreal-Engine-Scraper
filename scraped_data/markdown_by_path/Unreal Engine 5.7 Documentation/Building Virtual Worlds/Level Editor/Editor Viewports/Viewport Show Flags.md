# Viewport Show Flags

Descriptions of all of the available Show Flags for the viewports.

![Viewport Show Flags](https://dev.epicgames.com/community/api/documentation/image/6153e5cc-117b-4431-93d7-c231826352c3?resizing_type=fill&width=1920&height=335)

 On this page

[![Show Flag Header](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/602d0ae5-daed-467e-97e4-833b0f20b156/01-show-flag-header.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/602d0ae5-daed-467e-97e4-833b0f20b156/01-show-flag-header.png)

Click image for full size.

The Show Flags are found under the Viewport **Show** menu. They are a means by which you can show and hide many of the items that are viewed within the viewport. For example, you can hide all particles, all landscape objects, or something a bit more advanced like turning off all of the sprite icons.

[![Show Flag](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/204f6c59-8b23-49bb-b6f1-7557f22872ba/02-show-flag.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/204f6c59-8b23-49bb-b6f1-7557f22872ba/02-show-flag.png)

Click image for full size.
