# Editor Viewports

A guide to navigation techniques and visualization settings for the Unreal Editor viewports.

![Editor Viewports](https://dev.epicgames.com/community/api/documentation/image/87011606-2fec-4c7a-bd34-da269ebc6bd9?resizing_type=fill&width=1920&height=335)

[![Viewports Topic](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/4153acff-c469-4eb0-91f1-42a5dda09cf3/01-viewports-topic.png)](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/4153acff-c469-4eb0-91f1-42a5dda09cf3/01-viewports-topic.png)

Click image for full size.

The **Viewports** are your window into the worlds you create in Unreal. They can be navigated just as you would in a game, or can be used in a more schematic design sense as you would for an architectural Blueprint. The Unreal Editor viewports contain a variety of tools and visualizers to help you see exactly the data you need.
