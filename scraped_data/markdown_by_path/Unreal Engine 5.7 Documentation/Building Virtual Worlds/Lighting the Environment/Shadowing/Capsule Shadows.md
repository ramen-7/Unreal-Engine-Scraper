# Capsule Shadows

Information on using phsyics capsules for dynamic soft shadowing of Skeletal Meshes.

![Capsule Shadows](https://dev.epicgames.com/community/api/documentation/image/3f059173-cacd-445d-b95b-97dc0223beb2?resizing_type=fill&width=1920&height=335)

 On this page

**Capsule Shadows** can be used to enable support for soft shadow casting for Skeletal Meshes by using a capsule representation of your character made
from a Physics Asset. The soft shadows serve to ground the character in indirectly lit areas and give soft shadows in directly lit areas.
