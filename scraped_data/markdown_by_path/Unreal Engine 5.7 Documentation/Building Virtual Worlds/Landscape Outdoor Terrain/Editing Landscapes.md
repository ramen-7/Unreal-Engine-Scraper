# Editing Landscapes

Guide to using the tools for editing Landscape terrains.

![Editing Landscapes](https://dev.epicgames.com/community/api/documentation/image/f4cc9165-c0e5-4f6d-8bbc-4c7a03780926?resizing_type=fill&width=1920&height=335)

 On this page

The Landscape Editing tools enable you to create, shape, and refine environments directly within the editor. Using a combination of sculpting, painting, and layer-based workflows, you can design highly detailed terrains that support a wide range of gameplay and visual styles.

To utilize the Landscape Editor, first see [Creating Landscapes](https://dev.epicgames.com/documentation/en-us/unreal-engine/creating-landscapes-in-unreal-engine). Once a landscape is created, use the following tools to add detail:

* **Sculpt Mode**: Modify the terrain’s height using sculpting tools such as Sculpt, Smooth, Flatten, Ramp, and Erosion.
* **Paint Mode**: Apply and blend Landscape material layers to paint textures, colors, or surface types across the terrain.
* **Manage Mode**: Modify the size, shape, and and manage the landscape components. Add procedural landscape elements like Landscape Splines and Blueprint Brushes.
