# Unreal Editor Scripting Tutorials

Provides a list of example pages that show how to use Blueprints and Python to script and automate content production tasks in the Unreal Editor.

![Unreal Editor Scripting Tutorials](https://dev.epicgames.com/community/api/documentation/image/b472131e-f276-48b4-9962-ed9e35044a0a?resizing_type=fill&width=1920&height=335)

 On this page

The pages in this section show you how to use Blueprints and Python to automate some common content production tasks in the Unreal Editor. You can copy these examples directly, or use them as models for your own scripts.
