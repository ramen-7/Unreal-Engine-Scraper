UnrealBuildTool

UnrealBuildTool (UBT) manages the process of building Unreal Engine source code across a variety of build configurations.](/documentation/en-us/unreal-engine/unreal-build-tool-in-unreal-engine)
[![Unreal Header Tool](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/22258815-0139-4818-ab02-69ed6ae90e64/placeholder_topic.png)

Unreal Header Tool

UnrealHeaderTool (UHT) is a custom parsing and code-generation tool supporting the UObject system.](/documentation/en-us/unreal-engine/unreal-header-tool-for-unreal-engine)

Unreal Automation Tool

Unreal Automation Tool is a generic system used to automate processes, including testing and building games.](/documentation/en-us/unreal-engine/unreal-automation-tool-for-unreal-engine)
