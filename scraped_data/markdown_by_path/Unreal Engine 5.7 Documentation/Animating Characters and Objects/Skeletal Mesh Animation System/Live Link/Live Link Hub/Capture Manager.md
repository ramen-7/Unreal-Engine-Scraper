# Capture Manager

Learn how to use the Capture Manager layout in Live Link Hub.

![Capture Manager](https://dev.epicgames.com/community/api/documentation/image/7aa8fd5c-57b1-40a5-97b8-d796f3c3c0ed?resizing_type=fill&width=1920&height=335)

**Capture Manager** in [Live Link Hub](https://dev.epicgames.com/documentation/en-us/unreal-engine/live-link-hub-in-unreal-engine?application_version=5.5) is used to ingest footage captured on a variety of devices including audio, mono and stereo video, and mobile using the **Live Link Face** application. The footage is referred to as a take.

The **Capture Manager Layout** in Live Link Hub replaces the Capture Manager tool that was part of **Unreal Engine**. The original Capture Manager tool will be removed in a future version of Unreal Engine.
