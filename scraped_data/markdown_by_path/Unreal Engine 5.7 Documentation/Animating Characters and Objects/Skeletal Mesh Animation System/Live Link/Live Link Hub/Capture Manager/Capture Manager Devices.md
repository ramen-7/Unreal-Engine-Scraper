# Capture Manager Devices

Understand how to use the devices available in Capture Manager.

![Capture Manager Devices](https://dev.epicgames.com/community/api/documentation/image/cee306b0-f04e-4f66-a89b-47a7b61b2543?resizing_type=fill&width=1920&height=335)

**Capture Manager** provides a number of devices that determine how a take is ingested; **Mono Video**, **Stereo Video**, **Live Link Face**, and **Take Archive**. You should use the correct device for the captured data.
