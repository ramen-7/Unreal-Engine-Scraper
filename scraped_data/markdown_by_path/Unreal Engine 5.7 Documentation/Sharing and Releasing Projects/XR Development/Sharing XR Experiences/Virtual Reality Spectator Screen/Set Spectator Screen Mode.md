# Set Spectator Screen Mode

This node sets the Social (Spectator) Screen mode.

![Set Spectator Screen Mode](https://dev.epicgames.com/community/api/documentation/image/7fed6313-26ab-4d00-926d-d834650011f7?resizing_type=fill&width=1920&height=335)

![](https://d1iv7db44yhgxn.cloudfront.net/documentation/images/b5ab0143-ef8b-4cb4-8f24-9a22ef2826e2/setspectatorscreenmodenode.png)

This node sets the Social (Spectator) Screen mode.

**Inputs**

| Pin Location | Name | Description |
| --- | --- | --- |
|  | (In) Exec | Input execution pin. |
|  | Mode | The Social (Spectator) Screen Mode. |

**Output**

| Pin Location | Name | Description |
| --- | --- | --- |
|  | (Out) Exec | Output execution pin. |
